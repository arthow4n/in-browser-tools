Setup Instructions

To use this POC, you need to configure a Google Cloud Project:

  1. Go to the Google Cloud Console.
  2. Create a new project or select an existing one.
  3. Enable the Google Drive API in "APIs & Services" > "Library".
  4. Go to "APIs & Services" > "Credentials".
  5. Click "Create Credentials" > "OAuth client ID".
  6. Choose "Web application" as the Application type.
  7. Under "Authorized JavaScript origins", add the URL of this page (e.g., http://localhost:3000 or your deployed URL).
  8. Click "Create" and copy your Client ID.
